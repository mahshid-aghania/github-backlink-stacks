# Dentin Family Dentistry Guide

Official website: https://dfdentistry.ca/

This repository is a lightweight public resource hub for patients researching dental care in Vaughan. It summarizes common treatment categories, how to think about preventive care, and where to find detailed service information.

## Helpful links
- Main website: https://dfdentistry.ca/
- Services overview: https://dfdentistry.ca/services/
- Contact page: https://dfdentistry.ca/contact/
- Emergency dental care: https://dfdentistry.ca/services-2/emergency-dental-care/
- Oral surgery: https://dfdentistry.ca/services/oral-surgery/

## What this repo covers
- Preventive dentistry
- Restorative and cosmetic options
- Oral surgery and dental implants
- Patient education resources
- Ways to contact the clinic directly
