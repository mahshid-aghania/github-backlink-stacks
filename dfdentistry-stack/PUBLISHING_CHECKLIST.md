# Publishing Checklist for Dentin Family Dentistry

1. Create the three repositories in a real GitHub account you control.
2. Use the official site in the repo website field: https://dfdentistry.ca/
3. Add a concise repo description using natural language.
4. Upload the README and docs folder exactly once.
5. Add a small original image or logo only if you have rights to use it.
6. Connect related pages with relative markdown links inside the repo.
7. Do not clone the same copy into many throwaway repos.
8. Keep each repo useful, branded, and updated.
9. Add one or two relevant GitHub topics only.
10. Avoid automation or bulk-generated junk pages.
